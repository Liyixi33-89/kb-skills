## 0.0.1

首次发布。

- `kb-skills init` — 自动检测技术栈并生成 `kb-skills.config.ts`
- `kb-skills list` — 列出所有 Skill
- `kb-skills run <skill>` — 执行指定 Skill
- `kb-skills status` — 查看 KB 生成进度
- `kb-skills verify` — 完整性校验（CI 友好）
- `defineConfig` 辅助类型由 `@kb-skills/cli/config` 导出
